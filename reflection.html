Command Line Interface

What I learnt

Through this course, I have learnt the use of the Linux Command Line Interface and the command prompt in Windows. This was my first encounter with a command prompt and it took time for me to understand the purpose and the idea behind using the command prompt and not the other easy-to-use interfaces. I understood that the command prompt is a powerful tool to view or handle files. This is very useful in troubleshooting applications and operating systems itself; giving the user more control over the computer applications and operating system.

Practical session

During the practical session through Codio, I learnt the following:

  • The system commands, which include printing your current directory, navigating through and listing various directories.
  • Creating, opening, modifying and deleting directories and their content (the files) including hidden files.
  • Creating multiple directories and files of different types at different locations
  • Searching files and their contents
  • Managing various processes and installing packages

Knowledge gained and its application

I believe the knowledge gained through the lessons on command prompt will give me a different approach to computers and how I manage computer applications. During the course of the module, I was able to apply the knowledge gained to install Python and desired libraries through the command prompted. Specifically, I installed BeautifulSoup, Numpy, Pandas, Matplotlib, Seaborn and Openpyxl.

Moving forward I plan to practice the use of command prompt to master the platform and its functions.
